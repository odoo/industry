Property Assets & Distribution

This module centralizes the physical and financial lifecycle of property assets. It moves beyond simple records by establishing a dynamic link between the physical property, its chronological occupants (owners and tenants), and the Odoo Analytic Accounting engine.

Key Technical Capabilities

  • Unified Property Model: A data model designed to store essential physical attributes such as building, ground level, and surface area. It acts as a "Single Source of Truth" for the asset's physical characteristics.

  • Chronological Tenure Tracking: The system maintains a comprehensive history of both Owners and Tenants. By capturing start and end dates for every relationship, the module ensures full traceability of responsibility and occupancy over time.

  • Automated Analytic Lifecycle: To eliminate manual data entry, the module automatically generates a dedicated Analytic Account for every unique Property-Tenant pairing. This allows for granular financial tracking, enabling the precise allocation of bills and revenues to specific tenancies.

  • Weighted Distribution Engine: This feature provides a sophisticated ratio-based allocation system. It calculates distribution keys—initially derived from property surface areas—to define how communal expenses or building-wide bills should be split across multiple analytic accounts. These ratios remain fully editable to accommodate custom HOA or management agreements.
