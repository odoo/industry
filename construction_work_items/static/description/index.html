The Construction Work Item Module is not part of Odoo standard but uses Odoo Studio and leverages lines of python code; therefore installing this module may increase your subscription.


Purpose

Construction Work Item Module aims at enabling efficient quotation with a level of abstraction thanks to work items.

The module enables:

  • The management of work item templates related to products and combining products in various quantities to establish their price and margin;
    Those work items are typically provided into libraries by local actors to give a common referential.
  • The construction of quotations using work items behind order lines, with specific fine tuning to precise costs, price and margin.

This module uses service products with an invoicing policy set to manual delivered quantity.


Work items

A work item template consists in a custom model with:
  • A product to be used in sale order to leverage this work item;
  • A unit quantity and unit given the resulting work item size and counting unit;
  • A list of components, being goods and services, consisting in:
    • A component product;
    • Its unit cost, price, margin and unit;
    • The quantity of this component involved to lead to the work item product unit quantity.
  • The resulting work item cost, price and margin, computed from the sum of components cost and price in the appropriate quantities.

Work item templates

Work item templates are available from the Sales / Products menu.

The work item components data are replicated from the component products.

Work items behind sale order lines

When a sale order line points toward a product having a work item template, the work item template is duplicated and binded to the sale order line.
This local instance can be edited conveniently to fine tune the component cost, price and quantity depending on the quote context.

The work item binded behind a sale order line can be open click the line magic stick icon.

Products with work item templates can be used in quotation templates, which will generate local work items when using the quotation template in a quote.

Work item library import

To import a work item library, the herein file can be used as a referential:
  • Download the herein file.
  • Open the Product Variants list and proceed to import of the "1. Product Variants Import" sheet
  • Open again the Product Variants list, select them all and proceed to an export (import-compatible export) with only their names
  • Copy paste the export outcome in the "2. Product Variants Export" sheet
  • Open the Work Items list and proceed to import of the "3. Work Items Import" sheet
