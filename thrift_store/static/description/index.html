Odoo for Thrift Store

This industry package includes the necessary to run your thrift store business from employees planning to sales.

Basics

  • Point of Sale (POS): Flexible checkout system supporting sales by product, amount, or weight.
  • Employees: Manage staff and volunteer information.
  • Planning: Organize employee schedules and shifts with ease.

This industry relies only on standard Odoo features and does not include any customization.
