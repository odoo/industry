Odoo for Thrift Store

This Odoo industry package is designed to help second-hand shops and charity retailers run efficiently. It provides a flexible point-of-sale system capable of handling unique, one-off items and various pricing models like sales by weight. The solution also simplifies staff management, allowing you to organize schedules for employees and volunteers effortlessly. By streamlining these core operations, you can focus on maximizing the value of donations and serving your community.

Discover everything you need to know about this package by visiting the Knowledge App after installing it.

Basics

  • Point of Sale App: Process transactions quickly with a flexible checkout system that supports sales by product, fixed amount, or weight.
  • Employees App: Centralize information for all your staff members and volunteers in one secure directory.
  • Planning App: Organize shifts and schedules easily, ensuring you always have enough hands on deck for sorting and sales.
