Odoo for Libraries

This industry package includes all necessary settings for libraries to unify and optimize all core activities: encompassing efficient collection management, patron services, digital resource access, and the coordination of library events.

Basics

  • The Subscription App manage all members subscriptions and adapt pricing accordingly
  • The Point Of Sale App manage all rental activities, services fees and payments
  • The Return App manage all returns and monitor the delays
  • The Event App manage all in house events
  • The Website / Ecommerce App publish the online availability for the product catalog and inform about the library events

Customizations

  • Return App for all return management and customized settings
  • Route automation for the returns
  • Email templates for Delays
